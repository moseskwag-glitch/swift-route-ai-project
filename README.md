# \# SwiftRoute AI Project

# 

# \## Overview

# 

# SwiftRoute AI is an Artificial Intelligence project developed to analyze delivery shipment data and identify factors that influence whether a shipment arrives on time or late.

# 

# The project uses Exploratory Data Analysis (EDA) techniques to understand patterns within the dataset and support future machine learning model development.

# 

# \---

# 

# \## Objectives

# 

# \- Analyze shipment and delivery data.

# \- Identify factors affecting delivery performance.

# \- Visualize patterns using statistical graphs.

# \- Prepare data for machine learning applications.

# 

# \---

# 

# \## Dataset Features

# 

# The dataset contains information such as:

# 

# \- Warehouse Block

# \- Mode of Shipment

# \- Customer Care Calls

# \- Customer Rating

# \- Cost of Product

# \- Prior Purchases

# \- Product Importance

# \- Gender

# \- Discount Offered

# \- Weight in Grams

# \- Delivery Status (Target Variable)

# 

# Target Variable:

# 

# \- `Reached.on.Time\_Y.N`

# &#x20; - 0 = Delivered On Time

# &#x20; - 1 = Delivered Late
